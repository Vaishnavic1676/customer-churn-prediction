Customer Churn Prediction

This project predicts whether a telecom customer is likely to churn using machine learning.

Dataset:
Telco Customer Churn dataset

Technologies Used:
Python
Pandas
Scikit-learn
Matplotlib

Steps Performed:
- Data cleaning
- Feature encoding
- Handling missing values
- Logistic Regression model
- Class imbalance handling
- Model evaluation using ROC-AUC

Model Performance:
Accuracy: 74%
Recall (Churn): 78%
AUC Score: 0.84

Key Insights:
Customers with month-to-month contracts churn more.
Customers with longer tenure are less likely to churn.
Two-year contracts significantly reduce churn risk.

The trained model is saved using pickle for future predictions.
